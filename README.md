# 趋势分析回测结果展示

基于 OpenClaw Lens 构建的回测结果可视化平台。

## 功能

- 📊 回测指标展示（胜率、收益、夏普比率等）
- 📈 版本对比图表（v6.3 vs v6.5）
- 📝 优化说明文档
- 🔄 自动同步更新

## 技术栈

- React + Vite
- Tailwind CSS
- Recharts 图表
- GitHub Pages

## 自动同步

回测完成后自动运行：
```bash
./scripts/preview-sync.sh /path/to/backtest_summary.json
```
